########################################### Comment compiler le projet ###########################################

Exécuter la commande make dans le dossier contenant le fichier makefile.

Si cela ne marche pas :
	- Priez tous les dieux possibles
	- Compiler tous les fichiers à la main avec la commande : 

	- Répéter ces deux étapes jusqu'à ce que ça marche.

############################################ Utilisation du programme ############################################

Exécuter le programme avec la commande "./programme".

Il vous sera demandé d'entrer un nom de fichier. ATTENTION, l'extension n'est pas rajoutée automatiquement, il est donc de votre responsabilité de la préciser.
Si vous entrez le nom d'un fichier existant, il vous sera proposé d'ouvrir le fichier, il suffit de répondre par Y ou N, respectivement oui et non.
Une fois dans l'éditeur de texte, vous pourrez naviguer à l'aide des flèches directionnelles.

Pour quitter le programme, il suffit d'appuyer rapidement deux fois sur la touche echap (Esc sur votre clavier). Le fichier sera automatiquement sauvegardé et vous reviendrez au terminal.
